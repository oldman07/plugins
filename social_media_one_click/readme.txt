Easily add social media links to your WordPress site with just a shortcode.

== Description ==

Social Media One Click is a lightweight and intuitive plugin that enables users to add social media links to their WordPress site with minimal effort. The plugin offers an easy-to-use shortcode that can be customized to include links to multiple social media platforms.

**Features:**

- **User-Friendly Shortcode:** Use a simple shortcode to add social media links anywhere on your site. Customize the links for Facebook, Twitter, Instagram, and more.
- **Seamless Integration:** The plugin blends effortlessly with your existing WordPress theme, allowing you to apply custom styles or use the provided default styles.
- **Admin Panel Integration:** Manage your social media links directly from the WordPress admin panel, complete with a menu item and submenus for easy access.
- **Performance and Security:** Built with best practices to ensure optimal performance and security, safeguarding your site’s data.

**How It Works:**

1. **Install and Activate the Plugin:** Once activated, a new menu item titled "Social Media Links" will appear in your WordPress admin panel.
2. **Add Social Media Links:** Use the provided shortcode [social_media_links facebook="your_facebook_link" twitter="your_twitter_link" instagram="your_instagram_link"] in any post, page, or widget to display your social media links.
3. **Style and Customize:** The plugin includes a default stylesheet, but you can easily override the styles by editing your theme's CSS.
4. **Admin Management:** The plugin provides a settings page in the admin panel where you can manage your links and access additional submenus.

Enhance your website’s social media presence with Social Media One Click – a hassle-free way to integrate and display social media links on your WordPress site.
